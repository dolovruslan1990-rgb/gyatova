# Random Password Generator

**Автор:** Гятова Алина

## Описание

Приложение «Random Password Generator» — это графический генератор случайных паролей с возможностью настройки параметров и сохранения истории генерации.

## Функциональность

* Настройка длины пароля (4–30 символов) с помощью ползунка.
* Выбор типов символов: буквы, цифры, специальные символы.
* Генерация случайного пароля по заданным параметрам.
* Сохранение истории последних 50 сгенерированных паролей.
* Отображение истории с датой и временем генерации.
* Сохранение истории в JSON‑файл между запусками программы.

## Требования

* Python 3.6+
* Стандартные библиотеки Python (`tkinter`, `random`, `string`, `json`, `datetime`)
